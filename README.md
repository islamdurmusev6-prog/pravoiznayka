**Бесплатный юридический помощник для граждан России**

Получайте консультации по трудовым, жилищным, семейным и другим правам без дорогих юристов. Используйте AI для защиты своих прав!

## 🚀 Быстрый старт

```bash
# 1. Клонирование
git clone https://github.com/islamdurmusev6-prog/pravoiznayka.git
cd pravoiznayka

# 2. Установка
npm install

# 3. Конфигурация
copy .env.example .env.local
# Добавьте Groq API ключ в .env.local

# 4. Запуск
npm start
```

## 🔧 Установка

1. **Клонируйте репозиторий**
```bash
git clone https://github.com/islamdurmusev6-prog/pravoiznayka.git
cd pravoiznayka
```

2. **Установите зависимости**
```bash
npm install
```

3. **Получите Groq API ключи**
- Перейдите на https://console.groq.com
- Создайте аккаунт (бесплатно)
- Создайте 1-3 API ключа
- Скопируйте ключи

4. **Настройте переменные окружения**
```bash
copy .env.example .env.local
```

Отредактируйте `.env.local`:
```env
EXPO_PUBLIC_GROQ_API_KEY_1=gsk_...
EXPO_PUBLIC_GROQ_API_KEY_2=gsk_...
EXPO_PUBLIC_GROQ_API_KEY_3=gsk_...
```

5. **Запустите приложение**

Для iOS:
```bash
npm run ios
```

Для Android:
```bash
npm run android
```

Для веб-версии:
```bash
npm run web
```

Для физического устройства:
```bash
npm start
```
Отсканируйте QR-код в Expo Go

## API Интеграция

Приложение использует **Groq API** (бесплатно!) для генерации юридических консультаций.

### Преимущества Groq:
- ✅ **Полностью бесплатно** - нет платежей
- ✅ **Быстро** - ответы за 1-2 секунды
- ✅ **Надёжно** - 99.9% uptime
- ✅ **Масштабируемо** - поддерживает 100+ пользователей одновременно

### Лимиты:
- **Бесплатный план**: 30 запросов в минуту на один ключ
- **Несколько ключей**: 3 ключа = 90 запросов в минуту
- **С кэшированием**: +50% пользователей

### Как работает:
1. Приложение случайно выбирает один из 3 API ключей
2. Распределяет нагрузку между ними
3. Кэширует похожие вопросы
4. Экономит API запросы

### Получение API ключа

1. Перейдите на https://console.groq.com
2. Создайте аккаунт (бесплатно)
3. Перейдите в раздел "API Keys"
4. Создайте новый ключ
5. Скопируйте ключ в `.env.local`

Можно создать несколько ключей для лучшей масштабируемости!
